# AuRORA-doc
Documentação do software AuRORA

## Como rodar a documentação

### Pré-requisitos
- Python 3.8 ou superior
- pip (gerenciador de pacotes Python)

### Instalação
1. Clone o repositório:
   ```bash
   git clone <url-do-repositorio>
   cd AuRORA-doc
   ```

2. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```

### Executar a documentação
1. Navegue até o diretório da documentação:
   ```bash
   cd aurora-doc
   ```

2. Execute o servidor de desenvolvimento:
   ```bash
   mkdocs serve
   ```

3. Acesse a documentação no navegador:
   ```
   http://127.0.0.1:8000
   ```

### Comandos úteis
- `mkdocs serve` - Inicia o servidor de desenvolvimento
- `mkdocs build` - Gera os arquivos estáticos da documentação
- `mkdocs --help` - Exibe ajuda sobre comandos disponíveis
