**Foto:** (Placeholder para foto de uma desenvolvedora sênior)

Carla é uma desenvolvedora de software sênior de 35 anos com vasta experiência. Ela é mentora dos desenvolvedores juniores e é responsável pela arquitetura e qualidade dos projetos. Ela representa o grupo experiente de stakeholders.

### Demografia

*   **Idade:** 35
*   **Gênero:** Feminino
*   **Área de Atuação:** Desenvolvimento de Software / Arquitetura
*   **Experiência Profissional:** 12 anos

### Habilidades Técnicas

*   **Sistema Operacional:** Linux, macOS
*   **Linguagens de Programação:** Python, Java, JavaScript
*   **Frameworks:** Spring, Angular, React
*   **Bibliotecas:** Amplo conhecimento de várias bibliotecas
*   **Bancos de Dados:** PostgreSQL, MySQL
*   **Ferramentas:** Git (Avançado), Docker (Avançado)

### Objetivos

*   Garantir a manutenibilidade a longo prazo do projeto.
*   Criar uma ferramenta que seja fácil para os novos membros aprenderem e usarem.

*   Código mal escrito e de difícil manutenção.
*   Falta de padronização no processo de desenvolvimento.
*   Desenvolvedores juniores que não seguem as melhores práticas.
